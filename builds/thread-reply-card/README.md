# Build: thread-reply-card

Turn a "thoughts?" thread into a screenshot-ready **Steward card** — the
home's documented pattern for the user's "surprise me" social replies.

## Source
- Built this session after the user asked for a read on three X posts
  (Claude Projects course, Fable5 swarm, CodeWiki). The card format proved
  reusable, so it became a skill.
- Method: hand-built self-contained HTML (no image-gen — credits were locked;
  the HTML card is the reliable fallback). Includes a "copy reply" button.
- Frame as **THEIR stack**, not vendor promo (per user preference).
