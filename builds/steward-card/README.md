# Build: steward-card

A one-glance comparison card — **Claude Projects vs My Hermes Stack** — built
to fire back at a vendor-funnel thread. It makes the local-first, no-lock-in
case visually.

## Source
- Origin: a reply to an X post pitching "Claude Projects" as a paid course.
- Method: hand-built self-contained HTML (no external deps, no image-gen —
  credits were locked, so the HTML card was the reliable fallback).
- Rendered and verified clean (no clipped text / overflow) before docking.

## Files
- `index.html` — the card. Open in any browser. "copy reply" button copies the
  one-line rebuttal to clipboard.

## Principle check
- Local-first ✓ (no network, no vendor account)
- No lock-in ✓ (owns nothing of the user; documents exit)
- Transparency ✓ (this README states source + method)
- Yours ✓ (lives in the user's home repo)
